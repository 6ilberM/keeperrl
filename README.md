keeperrl
========

Source code of KeeperRL

Compiling
=========

-Prerequisites

  * make essentials
  * git
  * libsfml-dev 2+ (Ubuntu ppa that contains libsfml 2: http://www.playdeb.net/updates/Ubuntu/13.10#how_to_install )
  * freeglut-dev


In terminal:  
  ```
  git clone https://github.com/miki151/keeperrl.git
  cd keeperrl
  make -j 8
  ./keeper
  ```
